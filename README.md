# Akshat Siddh — Portfolio

A modern frontend portfolio and editorial platform focused on clean typography, smooth interaction, and structured storytelling. Designed with inspiration from contemporary product and publishing aesthetics.

## Overview

This project contains a developer portfolio . It is built as a frontend-only application with performance, readability, and motion consistency as core priorities.

The platform includes:

- Cinematic landing page
- About Projects
- About and timeline sections
- Contact interface
- Smooth animations and transitions
- Responsive design across devices

## Design System

The design system is built around typography, spacing rhythm, and restrained visual hierarchy.

### Typography

- Instrument Serif — display headings
- Inter — interface and body text
- JetBrains Mono — labels and metadata

### Visual Language

- Deep neutral palette
- Minimal borders and soft contrast
- Editorial spacing system
- Subtle grain and vignette textures

### Motion

- Smooth scrolling with Lenis
- Framer Motion transitions
- Minimal staggered animations
- Reduced-motion support
